WAMP is an open WebSocket subprotocol that provides two asynchronous messaging patterns: RPC and PubSub.

See the WAMP website for more info.

Overview

Note: clj-wamp currently supports HTTP Kit async servers only.

clj-wamp provides an interface layer for websocket communication using the WAMP subprotocol on the server-side. Compose a map of RPC and/or event callbacks and you're good to go.

It should work with any WAMP v1 enabled client. The examples on this site use AutobahnJS.

Installation

Add the following dependency to your project.clj file:

[clj-wamp "1.0.0-beta1"]
